# Personal Portfolio Website

A modern, responsive **personal portfolio website** built to showcase my skills, projects, and professional experience as a full-stack developer.

 **Live Website (Deployed on Render)**  
  
https://nomaantalibportfolio.onrender.com/
---

##  About the Project

This portfolio serves as a central platform to highlight my technical expertise, real-world projects, and development journey. It is designed with clean UI, smooth animations, and responsiveness to deliver a professional experience for recruiters and collaborators.

---

### Frontend
- HTML5
- CSS3 / Tailwind CSS

### Backend (if applicable)

- Fully responsive design (mobile, tablet, desktop)

## Commands Used

Below are the key commands used during development, testing, and deployment of this project:

###  Install Dependencies
```bash
npm install
```
### Run Development Server
```bash
npm run dev
```
### Build for Production
```Bash
npm run build
```
### Start Production Server
```Bash
npm start
```

